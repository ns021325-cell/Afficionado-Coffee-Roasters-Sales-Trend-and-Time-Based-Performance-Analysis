# Afficionado-Coffee-Roasters-Sales-Trend-and-Time-Based-Performance-Analysis



This project analyzes sales data for Afficionado Coffee Roasters to uncover trends, patterns, and time-based performance insights. The goal is to identify when sales peak, which periods underperform, and how customer behavior varies across time.


## 📂 Key business problems solved
• Identify overall sales trends across 2025
• Determine busiest and slowest days of the week
• Identify peak transaction hours
• Compare temporal demand patterns across store locations
• Support staff scheduling and operational planning
• Provide evidence-based explanations for observed patterns


## 🛠 Tools and Skills Used
• Python
• Exploratory Data Analysis 
• Matplotlib,Pandas,Numpy,Seaborn
• Streamlit Application


## 📌 Key Insights
1.	Morning hours(6-11 AM) consistently experience the highest sales.
2.	Fridays generate maximum revenue; Sundays are slowest
3.	Urban stores peak earlier, suburban stores peak later.
4.	Beverages dominate revenue; mid-day snacks offers promotional opportunities.
5.	Temporal patterns indicate staffing and operational adjustments can improve efficiency.
